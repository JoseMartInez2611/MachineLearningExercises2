# MachineLearningExercises2

Repositorio perteneciente a **Julián Guardián** y **José Martínez** donde se alojan los ejercicios prácticos del taller #2, solicitados por el docente en **Python** para la asignatura de **Machine Learning**.

## Contenido principal

- `solutions.py` → Código donde se encuentran las funciones solicitadas por el docente en el respectivo taller.
- `Numpy, Pandas, CSV` → Carpetas con los ejercicios y sus test individuales.
- `exercise.py` → Código donde se encuentra las funciones con los ejercicios (cada carpeta cuenta con su respectivo archivo exercise.py)
- `test.py` → Código donde se ejecuta las funciones creadas en el archivo exercise.py con los ejercicios respectivos de la carpeta (cada carpeta cuenta con su respectivo archivo test.py)
- `Data`→ Carpeta con archivos utilizados para los ejercicios de CSV 
- `OUTPUT`→ Carpeta donde se generan los archivos de resultado para los ejercicios de CSV que lo soicitan, contiene un archivo vacio para que git la reconozca 
## Cómo ejecutar

Simplemente ejecute el archivo solutions.py si desea imprimir las respuestas de todos los ejercicios de una vez en la terminal o ejecute por separado cada archivo test.py para impirmir los ejercicios individualmente. Al ejecutar cualquiera de estos 2 se imprimirá por consola un resumen de los datos de entrada de cada ejercicio y mostrará los datos de salida.
